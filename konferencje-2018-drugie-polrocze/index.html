Konferencje 2018 - drugie półrocze

7 czerwca 2018 · 2 min aby przeczytać
Agnieszka Gaworska

Coraz bliżej drugie półrocze 2018 roku zatem nadszedł czas, by przedstawić Wam zaplanowane na ten czas konferencje.

Czerwiec (uzupełnienie)​

15-16 czerwca, Dieburg, Niemcy – content strategy camp 2018

25-27 czerwca, Boston, Massachusetts, USA - An Event Apart Boston 2018

29-30 czerwca, Lizbona, Portugalia - Landing Festival 2018

23-25 lipca, Toronto, Kanada – ProComm 2018

24-25 lipca – IDEAS Online Conference 2018

30 lipca- 01 sierpnia, Washington DC, USA - An Event Apart Washington DC 2018

27-29 sierpnia, Chicago, Illinois, USA - An Event Apart Chicago 2018

28-31 sierpnia, Halifax, Nowa Szkocja, Kanada - DocEng 2018

Wrzesień​

4-7 września, Cleveland, Ohio, USA - content marketing world 2018

07 września, Londyn, Wielka Brytania - UX Conference 2018

09-11 września, Praga, Czechy - Write the Docs Prague 2018 W tym roku ponownie objęliśmy to wydarzenie patronatem medialnym :-)

10-12 września, Seattle, Waszyngton, USA - Best Practices Conference 2018

29 września, Warszawa, Polska - Konferencja Tłumaczy 2018

08-10 października, Orlando, Floryda, USA - An Event Apart Orlando 2018
